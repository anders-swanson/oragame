# Orasnake

Client-only Vite, TypeScript, and Phaser arcade snake game with bundled Oracle AI Database facts.

## Scripts

- `npm run dev` starts the local Vite dev server.
- `npm run test` runs pure game-logic and content validation tests.
- `npm run build` type-checks and builds the production bundle.
- `npm run preview` serves the built bundle.

The fact pack is bundled in `src/content/oracleFacts.ts` and points each topic back to the Oracle AI Database sample repo.
