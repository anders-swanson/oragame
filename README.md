# Ora-game Arcade

Browser based, arcade-style games to learn Oracle AI Database.

- [Oragame Arcade](https://anders-swanson.github.io/oragame/): arcade landing page.
- [orasnake](https://anders-swanson.github.io/oragame/snake/): inspired by the classic "snake" arcade game. Eat pickups and learn database facts.
