# Oragame: Interactive learning material for Oracle AI Database in playable game format

Each game is fun, instructive using browser-based frameworks to simply serve content to client users.

## Project Shape

Each subdirectory contains its own AGENTS.md file with specific instructions.

- arcade/ : homepage linking to each game
- orasnake/ : arcade game inspired by classic snake
 
## Development

- Use PhaserJS for game development
- Games should render well in both desktop and mobile viewports

## Adapt over time

As the project evolves, adapt the AGENTS.md in the root and subdirectories
